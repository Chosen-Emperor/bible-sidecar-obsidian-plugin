# Bible Sidecar Plus

A local-first Obsidian plugin that displays Bible books, chapters, and verses in a side-by-side view next to workspace notes, supporting text selections, annotation syncing, and offline searching.

## Language

**Scripture Passage**:
A segment of biblical text defined by a Book, Chapter, and Verse range.
_Avoid_: Quote, Bible verse, selection, reference text

**Bible Version**:
A specific translation of the Bible (e.g., KJV, ESV, or other translations).
_Avoid_: Translation, Bible API version, raw version

**Bible Sidecar**:
The workspace panel displaying the active Scripture Passage next to user notes.
_Avoid_: Sidebar, pane, BibleView, window

**Annotations**:
User-defined highlights (colors) and study notes associated with specific verses or phrases.
_Avoid_: Highlights, notes, markups, user notes

**Offline Cache**:
A local JSON database caching scripture texts downloaded on-demand or in bulk to enable offline reading and search.
_Avoid_: Local translation cache, local database, offline store

**Quick-Jump Grid**:
An overlay panel inside the Bible Sidecar displaying selectable lists of Books or Chapters for immediate navigation.
_Avoid_: Switcher panel, selector popup, jump menu
