# Release Notes - v1.3.16

This release adds mobile touch compatibility enhancements and the Quick Reference Launcher modal.

## New Features

### Quick Reference Launcher Modal
- Launched via command `Search and Reference Verse` or magnifying glass ribbon button.
- Type any verse reference (e.g. `Jn 3:16` or `Genesis 1`) to instantly see options: Open in Sidecar, Copy Text, Insert Link at Cursor, or Insert Text at Cursor.

### Mobile Navigation Layout
- Browse/Search tabs move to a native-feeling glassmorphic navigation bar at the bottom of the sidecar view on mobile screen sizes.

### Touch Interactivity
- Drag left or right to swipe navigation between chapters.
- Tap-to-select annotation mode with popup selector dot bar for quick mobile highlighting and note taking.
